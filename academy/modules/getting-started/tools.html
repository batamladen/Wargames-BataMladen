Tools for Wargames Challenges

To successfully complete the challenges in Wargames, you'll need to use a variety of cybersecurity tools. Each challenge may require different tools, such as network analyzers, password crackers, or forensic utilities.

Don't worry! All the tools you need for each challenge will be provided for download directly from the platform. Before starting a challenge, make sure to review the instructions, some challenges require GUI tools that are not downloadable on a ssh connection.

  • Tools are safe and commonly used in cybersecurity training.
  • Instructions for installation and usage will be included with each challenge.
  • If you have questions about a tool, check the FAQ or ask in the community.
