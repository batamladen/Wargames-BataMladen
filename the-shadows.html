The Shadows

Navigate through the shadows to uncover hidden secrets. Each level reveals a piece of the puzzle, with the flag from one level granting access to the next.
The password is only given for the Entery Point, you have to find the others yourself through the levels.

If you run into some issues or have ideas for new levels, please let me know!
